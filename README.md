# VehicHub
The Vehicle Rental System is a web-based platform designed to simplify and automate the process of renting vehicles online. The system allows customers to browse, select, and book vehicles such as cars and bikes through an intuitive online interface. It also includes a powerful admin panel that enables administrators to manage all aspects of the system efficiently.

Developed using PHP for server-side scripting, HTML/CSS for the user interface, JavaScript for interactivity, and MySQL for secure data storage, this system provides a complete and user-friendly digital solution for vehicle rental operations.
